Introduction to Python in UE 4

Posted on 2022-03-28 Edited on 2022-04-20 In Unreal Python

Python has been a widely used “glue” programming language in the media and entertainment industry. Most creative 3D production software provides Python API to help artists and developers optimize their production pipelines. Unreal Engine, a game engine developed by Epic Games, also allows developers to employ Python to improve their workflow. In this post, I will go through some background knowledge and setups necessary before you try to write and run your first Python script in Unreal Engine 4.

Background

What kind of problems could be solved by Python in UE4?

For the tasks you could accomplish with Python, I’d like to quote some words from the UE4’s official document:

  • Construct larger-scale asset management pipelines or workflows that tie the Unreal Editor to other 3D applications that you use in your organization.
  • Automate time-consuming Asset management tasks in the Unreal Editor, like generating Levels of Detail (LODs) for Static Meshes.
  • Procedurally lay out content in a Level.
  • Control the Unreal Editor from UIs that you create yourself in Python.

In short, Python is nearly a panacea except for writing the game logic itself. Python could successfully access most C++ APIs provided by UE4, which means it has almost the same power C++ has. At the same time, as a famous glue language, Python also has a large number of packages that could help developers complete some seemingly complicated works. For example, employing AI in animation automation with TensorFlow, managing your assets with highly visualized files like Xmind, or achieving tasks across different software.

Get Start

Enable plugin

Since Python is still an experimental feature in UE4, the Python Plugin in UE4 is not enabled by default. You can find the Plugins window in any UE4 project, and the Python Editor Script Plugin we need is under the Scripting section.

  • Edit > Plugins > Built-in > Scripting

Every Plugin under this section offers some APIs for the Editor, keeping all of them enabled as long as you may need them. Click their Enabled box and Restart the Editor to enable the Plugins. The version of the embedded Python in the Python Plugin is 3.7.7, and you can find it under your [UE4 main folder]\Engine\Binaries\ThirdParty\Python3\Win64.

You will also need the Output Log for future debugging. You can enable it in your UE4 Editor by choosing Window > Developer Tools > Output Log.

Set-up Your IDE

The last thing before we start is about the IDE, which is the Editor you will use in your future works. I recommend Microsoft Visual Studio Code since it is free and powerful enough to satisfy most scenarios. Other well-known Python IDE like PyCharm is also suitable for this task. For how to enable the autocomplete for UE4 Python scripting, you can find a detailed tutorial on UE4 documentation.
