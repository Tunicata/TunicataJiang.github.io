Check Asset Info

Every Struct and Object in Unreal Editor would have two kinds of properties, editor properties, and normal properties. The object and struct class both have two functions get_editor_property(name) and set_editor_properties(name, value, notify_mode=PropertyAccessChangeNotifyMode.DEFAULT). Those functions could get/set the value of any property visible to the editor, which is the nature of editor properties. Let’s continue to work on the texture.
